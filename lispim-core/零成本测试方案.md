## 使用步骤

### 1. 安装 Docker Desktop

下载地址：https://www.docker.com/products/docker-desktop/

### 2. 配置环境变量

```bash
# 复制环境变量模板
copy .env.example .env

# 或使用编辑器复制
# 确保 .env 文件在 lispim-core 目录下
```

### 3. 启动服务

**Windows:**
```batch
start-dev.bat
```

**手动启动:**
```bash
# 启动 Docker 服务
docker-compose up -d

# 等待服务就绪
sleep 10

# 启动 LispIM
sbcl --load lispim-backend-app.lisp --eval "(lispim-backend-app:main)"
```

### 4. 访问应用

- 网页访问：http://localhost:4321
- 测试账号：admin / admin123

## 升级到生产环境

当需要正式运营时，只需修改配置：

### 1. 购买云服务器

推荐配置：
- 腾讯云轻量应用服务器：~100 元/月
- 2 核 4G 5M 带宽

### 2. 配置真实服务

```bash
# 修改 .env

# 使用真实邮件
SMTP_HOST=smtp.qq.com
SMTP_PORT=587
SMTP_USER=your-email@qq.com
SMTP_PASSWORD=your-auth-code

# 使用阿里云短信
SMS_PROVIDER=aliyun
SMS_ALIYUN_ACCESS_KEY=your-key
SMS_ALIYUN_ACCESS_SECRET=your-secret

# 使用阿里云 OSS
OSS_ENDPOINT=oss-cn-shanghai.aliyuncs.com
OSS_ACCESS_KEY=your-key
OSS_SECRET=your-secret
```

### 3. ICP 备案

- 通过云服务商申请
- 免费，需 7-20 天

## 常见问题

### Q: Docker 容器启动失败？

A: 检查端口占用：
```bash
netstat -ano | findstr :5432
netstat -ano | findstr :6379
```

或重置 Docker：
```bash
docker-compose down -v
docker-compose up -d
```

### Q: 如何查看日志？

A:
```bash
# 查看所有容器日志
docker-compose logs -f

# 查看特定服务日志
docker-compose logs postgres
```

### Q: 如何重置数据库？

A:
```bash
docker-compose down -v
docker-compose up -d
```

这会删除所有数据！
